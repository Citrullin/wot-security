# WoT Security Best Practices

## Secure Transport

* HTTPS (HTTP + TLS 1.3) 
* CoAPS (CoAP + DTLS) 
* MQTTS (MQTT + TLS 1.3) 

## Authentication

* HTTPS with one of oauth2, basic, digest...
* CoAPS with one of psk, public, cert.
* MQTTS with basic (MQTT native username/password)
